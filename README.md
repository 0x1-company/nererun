# ネレルンとは

<img width="320px" src="./icon.png" />

ネレルンは、お子さんを対象とした「寝かしつけ」に特化した音声アプリです。

音声ストーリーは科学的根拠に基づき制作しており、なかなか寝付けないお子さんでも、眠りに誘うことができます。
